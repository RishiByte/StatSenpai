# StatSenpai

A lightweight web application for statistical exploration, visualization, and quick analysis — designed to help learners and practitioners inspect data and generate insights efficiently.

## Features

- Interactive data upload and preview (CSV/TSV)
- Common statistical summaries (mean, median, variance, percentiles)
- Visualizations: histograms, box plots, scatter plots, and line charts
- Quick correlation and basic regression tools
- Exportable reports and images for sharing results
